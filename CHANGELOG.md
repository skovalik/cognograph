# Changelog

## [0.1.0] - 2026-02-13

### Initial Public Release

Spatial canvas for AI workflow orchestration.

**Core Features:**
- **Context Injection** — Connected nodes automatically feed into AI prompts
- **10 Workspace Templates** — AI Research, Content Studio, Multi-Agent Pipeline, etc.
- **8 Node Types** — Conversation, Note, Task, Project, Artifact, Text, Action, Workspace
- **Multi-Provider AI** — Claude, Gemini, OpenAI support
- **Agent Mode** — Autonomous AI agents on canvas
- **Local-First** — No cloud, no accounts, your data stays on your machine

**Quality:**
- 655 tests (98.3% passing)
- 50% code coverage
- WCAG AA compliant
- TypeScript strict mode

**Known Issues:**
- Unsigned builds (expect security warnings)
- 11 depth-tracking edge case test failures (non-blocking)
