U1 — Splash Screen

Source: SplashScreen.tsx. Nodes appear as a constellation, edges connect them, then 5 different context flow paths cycle through — each lighting up different nodes.

Cognograph

Loading workspace...

Animation sequence:

  • 0.0s–0.7s: 10 rectangular nodes pop in with random size variation (±20%)
  • 0.7s–1.3s: Edges draw between connected nodes
  • 1.5s: "Cognograph" title fades in
  • 2.0s+: 6 flow paths cycle (18s total, 3s each) — final path lights up entire constellation from conversation node
  • Transitions: Nodes fade in/out smoothly (400ms), paths overlap so multiple can be partially visible
  • Randomization: Each load applies ±35px position jitter with collision avoidance (min 45px between nodes)

Implementation note: Use initSplashConstellation() on mount. Nodes that stay active between path changes maintain their glow continuously.
